# Sistem Manajemen Perpustakaan

## Deskripsi

Program ini merupakan aplikasi console berbasis Java yang digunakan untuk mengelola data buku pada perpustakaan sederhana.

Program dibuat untuk memenuhi tugas Mata Kuliah Pemrograman Berbasis Objek.

## Fitur

* Tambah Buku
* Lihat Semua Buku
* Cari Buku berdasarkan ID
* Cari Buku berdasarkan ISBN
* Update Buku berdasarkan ID
* Update Buku berdasarkan ISBN
* Hapus Buku berdasarkan ID
* Hapus Buku berdasarkan ISBN
* Validasi ISBN unik
* Validasi input angka
* Penyimpanan data ke file TXT
* Membaca data otomatis saat program dijalankan

## Konsep OOP

* Class dan Object
* Encapsulation
* Constructor
* Getter dan Setter
* ArrayList
* Method
* Exception Handling

## Struktur Project

src/

* Buku.java
* PerpustakaanManager.java
* Main.java

## Cara Menjalankan

Compile:

javac *.java

Run:

java Main
